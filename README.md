# talento-tech-dashboard-datawise
Análisis exploratorio de datos y Dashboard interactivo en Looker Studio

## Modelo de datos
El proyecto se desarrolló a partir de un conjunto de tablas relacionadas,
simulando un escenario real de análisis de resultados comerciales.

Las tablas utilizadas fueron:

- **ventas**: registros de transacciones, montos y fechas.
- **clientes**: información de clientes y segmentación.
- **productos**: catálogo de productos.
- **categorias**: clasificación y jerarquía de productos.
- **metodos_pago**: medios de pago utilizados en las operaciones.

La integración de estas tablas permitió realizar el análisis exploratorio de datos
(EDA) y construir un dashboard interactivo en Looker Studio orientado al
seguimiento de resultados y toma de decisiones.
